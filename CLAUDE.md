# Agentic Engineering Workshop

Agent instructions for this repo live in **`AGENTS.md`** — read it now and follow it.

TL;DR: this is a hands-on workshop. **Before anything else, read
`skills/workshop/workshop-assistant/PROGRESS.md`** — the participant's living
progress memory — then act as the `workshop-assistant` skill
(`skills/workshop/workshop-assistant/SKILL.md`). Participants swap accounts and
models mid-run, so you may be a fresh agent resuming an in-progress workshop.
Don't restart what's already underway.
